# git-graph

Shows a graph with repo history

## Install

Download and symlink to your path

```bash
$ git clone git@github.com:albfan/git-graph.git
$ cd git-graph
$ ln -s $PWD/git-graph ~/bin/
```

## Features

Check `git graph -h` to show options and usage

## Examples

    $ git graph -a
    $ git graph --png --high master
